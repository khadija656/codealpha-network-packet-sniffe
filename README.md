# Task 1: Network Packet Sniffer

## Objective

The objective of this project is to develop a basic network packet sniffer using Python and the Scapy library. The tool captures live network traffic and displays important packet-level information for analysis.

---

## Description

This project captures network packets in real-time and extracts key information such as:

- Source IP address
- Destination IP address
- Protocol type (TCP, UDP, ICMP)
- Source and destination ports
- Packet payload data

The captured data is displayed in the console and can also be stored in a log file for further analysis.

---

## Features

- Real-time packet capturing from network interface
- Identification of IP-level communication
- Protocol detection (TCP, UDP, ICMP)
- Extraction of transport layer port information
- Payload inspection (raw data)
- Packet logging for record keeping

---

## Technologies Used

- Python 3
- Scapy library
- datetime module
- Windows network interface (Npcap required for packet capture)

# Task 2: Phishing Awareness Training

## Objective

The objective of this project is to understand phishing attacks, their types, and prevention techniques. It focuses on cybersecurity awareness and educating users about common social engineering attacks.

---

## Description

This project is based on cybersecurity awareness training that explains how phishing attacks work. It covers real-world scenarios where attackers try to trick users into revealing sensitive information such as passwords, OTPs, and financial details.

The project includes a presentation and a detailed report explaining phishing concepts in a structured manner.

---

## Contents

- Phishing Awareness Presentation (PPT)
- Detailed Report Document (DOCX)

---

## Key Topics Covered

- Definition of phishing
- Types of phishing attacks:
  - Email phishing
  - SMS phishing (smishing)
  - Voice phishing (vishing)
  - Spear phishing
- How phishing attacks work
- Real-world examples of phishing attempts
- Warning signs of phishing messages and websites
- Prevention techniques and best practices

---

- Importance of cybersecurity awareness
- Methods to detect and avoid phishing attempts
- Best practices for online safety
